ABOUT GAME

What is the game?

LEID Party Game is, well as the name implies, a suite of party games! This project is created as a fun game that everyone can easily access and enjoy playing at anytime!

Which games can I play?

LEID Party Game Suite contains a variety of classic games which everyone can enjoy playing through! As of now the list of games contains: TicTacToe, Four-in-a-Row, Snap!, Breakout, Pong and a 2D race game we've decided to call "The Last Sausage Roll".

How do I play the game?

Follow the instructions found in the game and challenge your friends and determine the winner in various game modes like best of three, first to ten points or even singe match elimination!
